# Wine Quality Analysis

## Project Overview
This project performs exploratory data analysis (EDA) on a wine dataset to identify which chemical properties are associated with higher wine quality ratings.

## Dataset
The dataset contains physicochemical properties of wine such as acidity levels, sulfur dioxide content, pH, alcohol concentration, and quality ratings.

## Tools Used
- Google Sheets
- Descriptive statistics
- Correlation analysis
- Data visualization

## Analysis Steps
1. Calculated descriptive statistics for all variables.
2. Compared chemical properties between good and bad wines.
3. Ranked features based on mean differences.
4. Conducted correlation analysis with wine quality.
5. Visualized relationships between variables.

## Key Findings
- Alcohol shows the strongest positive relationship with wine quality.
- Volatile acidity has the strongest negative relationship with quality.
- Sulphates and citric acid show moderate positive associations.
- Residual sugar and pH have minimal influence on wine ratings.
